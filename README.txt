Hello, Welcome to MangaScroller

To get this working for Windows, follow the steps below.

1. Install Python3
	-run the python-3.9.9-amd64.exe in the folder
	-install to default location
	-make sure to check the "Add to PATH" checkbox
2. Install Selenium
	-open command prompt
	-run "pip install selenium"
3. double click "MangaScroller.py" and paste in your link. W and S control the scroll speed of the program.
   Program will autoclose when it reaches the bottom
